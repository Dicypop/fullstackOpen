# Navigating to the single page app

```mermaid
sequenceDiagram
    participant user
    participant browser
    participant server

    user->>browser: Navigate to https://studies.cs.helsinki.fi/exampleapp/spa
    browser->>server: GET /exampleapp/spa
    activate server
    server-->browser: SPA HTML page
    deactivate server

    browser->>server: GET /exampleapp/main.css
    activate server
    server-->>browser: main.css
    deactivate server

    browser->>server: GET /exampleapp/spa.js
    activate server
    server-->>browser: spa.js
    deactivate server

    Note right of browser: The browser starts executing spa.js

    browser->>server: GET /exampleapp/data.json
    activate server
    server-->>browser: JSON data [{ "content": "Example note", "date": "2024-07-30" }, ...]
    deactivate server

    Note right of browser: JavaScript dynamically generates and inserts notes into the page
