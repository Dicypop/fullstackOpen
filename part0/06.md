# New note on single page app diagram

```mermaid
sequenceDiagram
    participant user
    participant browser
    participant server

    user->>browser: Enter note and click Save
    browser->>browser: JavaScript captures the form submission
    browser->>browser: Prevent default form submission
    browser->>server: AJAX POST /new_note_spa with JSON data
    activate server
    server-->browser: 201 Created
    deactivate server

    Note right of browser: JavaScript updates the UI to display the new note without reloading the page
